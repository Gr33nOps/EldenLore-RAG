# EldenLore RAG: The Compendium of Grace

EldenLore RAG is an AI-powered Retrieval-Augmented Generation tool that lets you ask questions about Elden Ring lore and get accurate answers grounded in a custom database of texts. It uses ChromaDB for semantic search, Groq for language generation, and offers multiple character personas to answer in distinct voices.

## Features

- Semantic search across a large collection of Elden Ring lore
- Context‑aware answers generated by Groq LLMs
- Multiple character personas (e.g., Ranni, Melina, Gideon)
- Interactive lore quizzes to test your knowledge
- Web scraping tools to expand the lore database
- Clean web interface powered by Streamlit

## Quick Start Guide

### 1. Clone the repository

```bash
git clone https://github.com/Gr33nOps/EldenLore-RAG.git
cd EldenLore-RAG
```

### 2. Install Python (if not already installed)

- **Windows**: Download the installer from [python.org](https://www.python.org/downloads/) and run it. **Make sure to check "Add Python to PATH"** during installation.
- **macOS**: Use Homebrew – `brew install python` or download from python.org.
- **Linux (Debian/Ubuntu)**:
  ```bash
  sudo apt update
  sudo apt install python3 python3-venv python3-pip
  ```

Verify the installation:

```bash
python --version   # or python3 --version
```

### 3. Create and activate a virtual environment

```bash
# Windows
python -m venv venv
venv\Scripts\activate

# macOS / Linux
python3 -m venv venv
source venv/bin/activate
```

### 4. Install dependencies

```bash
pip install -r requirements.txt
```

### 5. Set up your Groq API key

Create a `.env` file with your key using this single command (replace `your_actual_groq_api_key` with your real key):

```bash
echo "GROQ_API_KEY=your_actual_groq_api_key" > .env
```

If you don't have a Groq API key, get one from [console.groq.com](https://console.groq.com).

### 6. (Optional) Scrape lore from the web

If you want to collect the latest lore texts, run:

```bash
python scraper.py
```

This will populate the `data/lore/` folder with text files. You can skip this step if you already have lore files in that directory.

### 7. Ingest the lore into the vector database

This step builds the ChromaDB index so the app can search efficiently. Run:

```bash
python ingest.py
```

**Important:** Run this once before launching the app for the first time, and again whenever you add or modify lore files.

### 8. Launch the web app

```bash
streamlit run app.py
```

Open your browser and go to [http://localhost:8501](http://localhost:8501). You can now ask questions about Elden Ring lore.

## Troubleshooting

**ModuleNotFoundError** – Make sure your virtual environment is active and all dependencies are installed:

```bash
pip install -r requirements.txt
```

**Streamlit fails to start after moving the project** – Recreate the virtual environment:

```bash
rm -rf venv          # or rmdir /s venv on Windows
python -m venv venv
pip install -r requirements.txt
```

**The AI gives unrelated answers** – The vector database may be missing or outdated. Run `python ingest.py` again.

**Scraping fails or takes too long** – Some websites may change their structure. You can skip scraping and manually add your own `.txt` files to `data/lore/`.
